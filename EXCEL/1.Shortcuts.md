1. F2 :- Edit Cell
2. Ctrl + Space :- Select Column
3. Shift + Space :- Select Row
4. Ctrl + U :- Underline
5. C + Entre or Esc :- Stop Writing In The Cell
6. Ctrl + R :- Copy And Paste in Horizontally Selected Cell
7. Ctrl + D :- Copy And Paste in Vertically Selected Cell
8. Ctrl + F :- Find
9. Ctrl + H :- Replace
10. F7 :- Spell Check
11. Ctrl +1 :- Format Cell
12. F4 :- For Absolute Referencing E.g. $A$3
