# Sliding Window Algo: SUPERSET

arr = [.............i.........j............]

- Both Pointers are movable in either directions
- Array can be sorted or unsorted
- i and j - can be any indices, not necessarily the first and last index
- Window: Values between i and j indices
- Window - Fixed or Variable
- Fixed: j-i: Diff: Constant
- Variable Sliding Window: j-i: Diff: Variable



# Two Pointers: SUBSET

arr = [i..................j]
	  start               end


- Both Pointers are movable in nature
- i: L to R, j: R to L	
- Array should be sorted
- i and j - necessarily end indices
