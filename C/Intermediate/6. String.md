
→ A string is a 1-D array of character terminated by a null ('\0').
	Example
```c
char string[]={'S','T','R','I','N','G','\0'};

```

→ String can also be initialization as
```c
char string[]="STRING";
```

→ `scanf` and `printf`
```c
scanf("%s",string);
printf("%s",string);
```

## gets() and puts()

`scanf()` and `printf()` is not capable for multi-word string. 
	The `gets()` and `puts()` can be used for multi-word string

```c
gets(string);
puts(string);
```
.
	`gets()` and `put()` can only work with only 1 string
