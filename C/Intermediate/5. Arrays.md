# Simple Array
→ An array is a collection of similar element.

**Array Declaration and Initialization
```c
int array1[30];
int array2[]={2,4,12,5};
```

**Accessing Elements of an Array**
```c
scanf("%d",&array1[i]); //store data in array
a=array1[i]; //read data from an array
```
i → (0,n-1) 
n:- no. of element

**Pointer of Array**

```c
&array[0] == array
&array[1] == array+1
```

### Pointer (to an element of array)

```c
array==&array[0][0]

array[i]=&array[i][0]
array[i]+j=&array[i][j]
((array+i)+j)=&array[i][j]
```
