# Array to Pointer to String

```c
char *pstr[]={
"STRING1"
"STRING2"
"STRING3"
}
```

Reason to use an Array of Pointer to String
	- Efficient use of memory
	- To obtain greater ease in manipulation of the string

Limitation of Array of Pointer to String
	- We cannot receive the string from keyboard
