
- If a condition is **True**
	Output $\neq$ 0
- If a condition is **False**
	Output = 0

Therefore,
	`if (i=5)`
Since, the output is 5($\neq$ 0), the condition will be deemed to be true and the if statement will be executed

## Logical Operator
1.  && → And
2. || → Or
3. ! → Not
	E.g. `!(3>4)` will be true
