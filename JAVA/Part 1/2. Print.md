The print command `System.out.println("Hello world");` prints the text "Hello world".

# println (adds a line break )

```java
System.out.println("Hello ");
System.out.println("world!");
```

```Output
Hello 
world!
```

# print (does not add a line break)

```java
System.out.println("Hello ");
System.out.println("world!");
```

```Output
Hello world!
```
